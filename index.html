DOM中的事件

DOM事件

事件流：描述的是从页面接受事件的顺序

事件冒泡：事件最开始最具体的元素接收，然后逐级向上传播

事件捕获：不太具体的节点最早接受到事件，最具体的节点最后接收到事件

HTML事件处理：直接将事件添加到HTML结构中
btn.onclick=”function(){}”

DOM0级事件处理：把一个函数赋值给一个事件的处理程序属性
btn.event = function(){};
btn.event = null;

DOM2级事件处理：

addEventListener()和removeEventListener(),接受三个参数：处理事件名，处理程序函数，布尔值(ture表示事件捕获，false表示事件冒泡),事件名前面不加on

btn.addEventListener("click",f,flase)    //绑定事件
btn.removeEventListener("click",f,flase) //解除事件

IE事件处理程序：attachEvent(),detachEvent(),接受两个参数：处理事件名，处理程序函数； 事件名前面加on

btn.attachEvent("onclick",f)
btn.detachEvent("onclick",f)

使用DOM0进行事件绑定，一个事件只能绑定一个函数，如果再次绑定别的函数会将前一个覆盖，事实上同一个事件支持绑定多个函数。而用DOM2级和IE事件处理可以进行多次绑定，比如同一个节点的click事件绑定多个处理函数

事件对象

当我们给事件绑定一个函数时，我们会将事件的信息（如click操作的绑定目标，事件类型等）作为一个对象传入函数，即绑定事件的事件的函数有一个默认的传入参数event，在IE的早期版本event对象要进行兼容

//传入函数
function(event){  //event为添加的事件对象（参数名起为其他也是同样作用）
    event = event || windows.event //IE中的event兼容
  console.log(event.target)}   //显示事件目标

事件代理

我们可以使用事件冒泡来实现事件代理，即我们要给一个父节点的所有子节点绑定同一事件时，我们不用遍历的方法给所有子节点都绑定事件，而是只给父节点绑定事件，这样子节点进行触发，冒泡到父节点，照样可以触发该事件。期间我们还可以对子节点进行筛选，只绑定需要绑定的子节点

当使用事件代理时，target指的是被触发的事件节点，而currentTarget指的是始终是绑定事件的父节点。如果没有事件代理，则target和currentTarget相同

事件触发

eventTarget.dispatchEvent(type)

事件节点.dispatchEvent("click")

兼容IE：fireEvent(e)

阻止事件冒泡和默认行为

function x(e){
    e=e||window.event

    e.stopPropagation()||event.cancelBubble=true;   //阻止事件冒泡

    e.preventDefaule()||event.returnValue=true;  //阻止事件默认行为（如阻止链接跳转，阻止文字被选定)         

    e.stopImmediatePropagation()//阻止事件冒泡，并阻止该节点后续事件的触发
}

事件属性和方法

||后面的都是对IE进行兼容

event || windows.event 事件对象

type属性 获取事件类型

target属性||srcElement属性 获取事件目标

stopPropagation() 方法 || cancelBubble属性(true为阻止,event.cancelBubble=true） 阻止事件冒泡

preventDefaule()方法 || returnValue属性(true为阻止,event.returnValue=true） 阻止事件默认行为(例如阻止a标签的跳转）

e.stopImmediatePropagation()//阻止事件冒泡，并阻止该节点后续事件的触发

关于JS中的prototype

原型继承

JS中的一切皆为对象，而每个对象中都有一个prototype属性，称为原型对象。

利用原型继承我们可以在JS中实现其他语言中“类”

在创建构造函数时，使用构造函数创建的对象也有一个原型对象，这个原型对象指向构造函数的原型对象，当调用创建对象的方法时，先查找自身，如果没有，则从自身原型对象里查找，还没有，就从构造函数的原型对象中查找，还没有就继续向上查找。这就叫做原型继承。而原型之间逐级向上的关系则称为原型链
