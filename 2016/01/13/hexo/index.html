hexo搭建个人博客

利用hexo可以在github搭建个人博客

1.安装Node和git

https://nodejs.org/en/

https://desktop.github.com/

2.安装hexo

(1)打开git 输入

$ npm install -g hexo

(2)在任意路径新建blog文件夹，并用git进入该文件夹

(3)在该文件夹下输入

$ npm install

$ hexo g

$ hexo s

(4)安装成功，在浏览器输入 localhost:4000 查看效果

3.github搭建项目

在github搭建名为username.github.io的项目(username就是github的用户名）

4.创建ssh

(1)git输入

$ ssh-keygen -t rsa -C "邮件地址@youremail.com"

期间会要求输入password，输入字符不会显示出来，如果输入则每次上传时都会要求输入，不输入则按回车继续

(2)添加SSH Key到GitHub

把生成的id_rsa.pub里信息复制到在github的setting里的ssh中

到个人邮箱确认github发的确认邮件

(3)ssh连接测试

git输入 $ ssh -T git@github.com

(4)设置用户信息

git输入

$ git config --global user.name "用户名"

$ git config --global user.email  "邮件地址@youremail.com"

成功信息如下：

Hi cnfeat! You’ve successfully authenticated, but GitHub does not provide shell access.

5.更改_config.yml

在blog文件夹中找到_config.yml，修改该文件

在deploy下添加

type: git
repository: https://github.com/HeGangSZ/HeGangSZ.github.io.git(从自己的github的repository中复制地址）
branch: master

冒号后有个空格不要输少了！否则会错误

6.更新上传

在git下输入

$ npm install hexo-deployer-git --save 

$ hexo g

$ hexo deploy

更新完毕后进入https://用户名.github.io/即可查看个人博客

以后每次更新用git进入blog目录输入

$ hexo g

$ hexo deploy
