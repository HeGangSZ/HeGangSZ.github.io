关于JS中的prototype

  • javascript

原型继承

JS中的一切皆为对象，而每个对象中都有一个prototype属性，称为原型对象。

在创建构造函数时，使用构造函数创建的对象也有一个原型对象，这个原型对象指向构造函数的原型对象，当调用创建对象的方法时，先查找自身，如果没有，则从自身原型对象里查找，还没有，就从构造函数的原型对象中查找，还没有就继续向上查找。这就叫做原型继承。而原型之间逐级向上的关系则称为原型链

xiaoming ----> Student.prototype ----> Object.prototype ----> null

如果将x方法加到构造函数prototype里，则所有创建的新对象所使用的x方法都是同一个。
如果不加到构造函数prototype中，而直接在构造函数中定义x方法，则每个创建的新对象都会创建一个x方法，相互之间的x方法是不同的

//创建一个构造函数，并在其原型添加方法
function Test(name,age){
    this.name = name;   //构造函数需要先将属性传进来赋予给this，this指向新创建的对象
    this.age = age;
}     //调用属性时先调用自身属性，如没有则从原型上找，还没有就顺着原型链继续往上找该属性
Test.prototype={   //prototype指的是构造器的原型对象，通过构造函数创建的对象共用来自这个原型对象的方法和属性
    hello:function(){
        alert("hello"+this.name+",your age is"+this.age)
    }
};

ES5中Object.creat()方法

var bill = Object.creat(teacher)相当于直接将teacher作为构造函数创建一个bill对象

Class继承

如果需要将一个构造函数进行扩展，创建一个新的构造函数，这在别的语言中相当于类的继承，而在JS中没有类的概念，所以我们要通过原型继承来实现这一功能

ClassA为父类，现需创建ClassB为子类

(1)

function ClassB(){

    ClassA.apply(this)
}
//首先调用ClassA的方法，使this指向ClassB，此时ClassB可以调用ClassA的方法，但原型链没有继承过来，而不能调用ClassA原型中的方法；

(2)

function F（）{}
//创建一个空函数来桥接原型链

(3)

F.prototype = ClassA.prototype 
//使ClassA的原型赋给F的原型，这样F和ClassA的原型和原型链是一样的

(4)

ClassB.prototype = new F()
//创建一个新的F对象来赋给ClassB.prototype，而F对象的原型又是ClassA的原型，则此时ClassB.prototype指向ClassA.prototype，则ClassB可以调用ClassA原型中的方法

ClassB.prototype  ----> ClassA.prototype 

此时在ClassB.prototype中增加属性相当于在F对象增加属性，并不会影响到ClassA.prototype

(5)

ClassB.prototype.constructor = ClassB
//在上一步中使ClassB.prototype.constructor指向了F()，这里进行恢复

下面将这一步骤进行封装

function inherits(Child, Parent) {
    var F = function () {};
    F.prototype = Parent.prototype;
    Child.prototype = new F();
    Child.prototype.constructor = Child;
}

这个inherits()函数可以复用：

function Student(props) {
    this.name = props.name || 'Unnamed';
}

Student.prototype.hello = function () {
    alert('Hello, ' + this.name + '!');
}

function PrimaryStudent(props) {
    Student.call(this, props);
    this.grade = props.grade || 1;
}

// 实现原型继承链:
inherits(PrimaryStudent, Student);

// 绑定其他方法到PrimaryStudent原型:
PrimaryStudent.prototype.getGrade = function () {
    return this.grade;
};
